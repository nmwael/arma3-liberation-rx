Liberation RX + RHS Client Modpack

Import in Arma 3 Launcher → Presets → Import → Paste URL or save as .html and import file

Voice: Discord (no TFAR required)

Required Mods (Subscribe to all)

  • CBA_A3 - Framework (Required)
  • ACE3 - Medical, ballistics, interaction
  • CUP Terrains Core - Map base
  • CUP Terrains Maps - Altis, Tanoa, Chernarus, etc.
  • CUP Units - Factions
  • CUP Weapons - Weapons
  • CUP Vehicles - Vehicles
  • ACE Compat - CUP - ACE + CUP compatibility
  • Enhanced Movement - Climbing, vaulting
  • Advanced Towing - Vehicle towing
  • Advanced Rappelling - Heli rappelling
  • Advanced Urban Rappelling - Urban rappelling

RHS Mods (Free, Full Factions - Highly Recommended)

  • RHS: AFRF - Russian Federation Armed Forces
  • RHS: USAF - United States Armed Forces
  • RHS: GREF - Serbia, Yugoslavia, others
  • RHS: SAF - Serbian Armed Forces
  • ACE Compat - RHS - ACE + RHS compatibility

Optional (Client Only - Quality of Life)

  • ShackTac UI (STUI) - Better HUD, stance, stamina
  • Blastcore Edited - Visual effects
  • JSRS Soundmod - Sound overhaul (heavy)
  • Enhanced Soundscape - Lighter sound mod

Liberation RX Mission

Mission is server-side only. No client subscription needed.

Maps: Altis (default), Tanoa, Livonia, Weferlingen, Ravenport

Liberation RX Faction Settings (Server Config)

With RHS installed, set in mission parameters:

  • BLUFOR: RHSUSAF / RHSGREF / RHSSAF
  • OPFOR: RHSAFRF / RHSGREF
