Manual Installation

  1. Open Arma 3 Launcher → Mods tab
  2. Click + (Add Mod) → Paste Workshop URL or search by name
  3. Click Subscribe for each required mod
  4. Wait for downloads to complete
  5. Launch Arma 3 → Multiplayer → Connect to server IP
